public key:

Aliases:
git config --global alias.<shortcut> <how you want shortcut> 
EX1: git config --global alias.st status => ( git status <=> git st ) 
EX2: git config --global alias.cm "commit -m" => ( git commit -m "test" <=> git cm "test" )
